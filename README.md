# C# Beginner Game

オブジェクト指向学習用のコンソール RPG バトルひな型です。

詳細と学習の進め方は [RpgBattle/README.md](RpgBattle/README.md) を参照してください。

## クイックスタート

**推奨: Visual Studio 2022** で [`RpgBattle/RpgBattle.sln`](RpgBattle/RpgBattle.sln) を開く → Ctrl+F5 で実行。

Visual Studio 2017 / Express では正しく開けません（クラスファイルが表示されない・ビルドできないことがあります）。

コマンドラインの場合:

```bash
cd RpgBattle
dotnet run
```

※ .NET 8 SDK が必要です。
